File String Manipulation
^^^^^^^^^^^^^^^^^^^^^^^^


New in v1.01
 - GetFileNoExtension > Returns filename without extension
 - Terrible bug fix in the GetFilePath
 - Changed example from File Thingy to File String Manipulation

Some function to manipulate file strings.
 - GetFileName > Returns only the filename
 - GetExtension > Returns file extension (optional lowercase)
 - GetPath > Returns only path. (optional without drive)
 - GetDrive > Returns drive (optional include "\")
